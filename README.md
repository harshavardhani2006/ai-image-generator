# 🎨 AI Image Generator

This application generates AI-powered images from text prompts. Users can enter a description of the image they want, provide their Hugging Face API key, and generate high-quality images using the FLUX.1-dev model. The app also allows users to view the Python code used for image generation, making it useful for both image creation and learning purposes.

### AI Model Used
- **FLUX.1-dev** by Black Forest Labs

### Why I Chose It
- Produces high-quality images from text prompts.
- Supports a wide variety of creative and realistic image styles.
- Easy to integrate using the Hugging Face Inference API.
- Fast and reliable for building AI-powered applications.

### Technologies Used
- Python
- Gradio
- Hugging Face Hub
- FLUX.1-dev
- PIL (Python Imaging Library)


## How can someone run the app locally?

### Clone the Repository

```bash
git clone https://github.com/harshavardhani2006/ai-image-generator.git
cd ai-image-generator
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```text
http://127.0.0.1:7860
```
